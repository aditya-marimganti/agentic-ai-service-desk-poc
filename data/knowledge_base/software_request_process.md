# Software Request Process

To request new software or a license for an existing tool, submit a request through the internal Service Portal under the "Software Requests" category. Include the name of the software, the business justification, and your manager's name for approval.

Standard software requests (tools already approved for company-wide use, such as Slack, Zoom, or Microsoft Office add-ons) are typically approved within 1-2 business days and do not require additional security review.

Requests for new or unapproved software that has not been used elsewhere in the company will go through a security and compliance review, which can take 5-10 business days. This includes checking the vendor's data handling practices and ensuring compatibility with company systems.

Once approved, software will either be automatically pushed to your device through the company's device management system, or you will receive an email with installation instructions if manual installation is required. If software is not installed within 48 hours of approval, contact the IT helpdesk to check on the deployment status.
